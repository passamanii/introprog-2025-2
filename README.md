Exercícios de Python

Este repositório contém exercícios desenvolvidos durante meu processo de aprendizagem em Python.
O objetivo é praticar lógica de programação, estruturas de controle, funções e resolução de problemas matemáticos.


Conteúdo

- Estruturas condicionais (if, else)

- Laços de repetição (for, while)

- Funções
- Cálculo de média
- Busca em listas
- Intervalos numéricos
- Cálculo de área
- Série de Taylor
- Fatorial


Objetivo

- Consolidar os fundamentos da linguagem Python através de prática constante e resolução de exercícios.


Tecnologias utilizadas:

- Python 3
- VSCode
